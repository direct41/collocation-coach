# Collocation Coach Product Plan

## Product

Telegram bot that helps Russian-speaking learners sound more natural in English through short daily collocation lessons.

## Target User

Primary segment:
- learners with level A2-B2
- already study English regularly
- understand many words but combine them unnaturally

## MVP Promise

Every day the bot sends a 2-3 minute lesson with:
- 3 new collocations
- 2 review collocations
- short explanation in Russian
- one correct example
- one common mistake
- one quick practice step

## MVP Success Signal

The launch is good enough if early users:
- complete the first lesson
- return during the first week
- say the format is short, clear, and useful

## Product Phases

### Phase 1: Definition
- freeze lesson format
- define commands and settings
- prepare content template

### Phase 2: Prototype
- build working Telegram bot
- support onboarding and one lesson flow
- load sample content

### Phase 3: MVP
- add daily delivery
- add review loop
- add 30 days of content

### Phase 4: Public Release
- improve setup docs
- add basic tests
- publish open-source repository

## Key Risks

- weak content quality
- too much lesson length
- poor review timing
- difficult self-hosting

## What Is Needed From The Project Owner

- preferred tech stack if any
- GitHub account or organization name
- Telegram bot token
- preferred default timezone and launch language
- decision on open-source license if not MIT
