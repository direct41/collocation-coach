# Content Structure

Lesson content is stored in YAML files under level-band folders.

Initial layout:

- `content/lessons/a2_b1/`
- `content/lessons/b1_b2/`

Each file represents one lesson unit and contains exactly 3 new collocation items.

Example:
- `content/lessons/a2_b1/day-001.yaml`

Review items are generated from user progress and are not hardcoded into the lesson files.
